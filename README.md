# tier1-emergence-papers

Papers, figures, supplementary material, and replication configs for the Tier‑1 Emergence research program. This repository contains the theoretical foundations, experimental designs, and reproducible artefacts that accompany the simulation work in `tier1-emergence`.

---

## Contents

- **Two‑Tier Emergence Hypothesis**  
  Intelligence as first‑order emergence; consciousness as second‑order emergence.  
  Manuscript, notes, and supporting materials.  
  `/TwoTierEmergence/`

- **Figures and Diagrams**  
  Visualisations, conceptual schematics, and experiment‑ready graphics.  
  `/figures/`

- **Replication Configurations**  
  Parameter sets, environment definitions, and experiment metadata for reproducing results.  
  `/replication/`

- **Bibliography**  
  Shared reference library for all papers in this research program.  
  `references.bib`

---

## Purpose

This repository serves as the scholarly layer of the Tier‑1 Emergence project.  
Where `tier1-emergence` provides the simulation engine, graph substrates, and analysis tools, this repository provides:

- theoretical frameworks  
- formal definitions  
- experimental protocols  
- manuscripts and drafts  
- replication instructions  

Everything required to understand, reproduce, and extend the research.
